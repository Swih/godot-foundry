# Contributing to godot-foundry

Thanks for your interest!

## How to contribute

1. Fork the repo
2. Create a branch for your feature
3. Make your changes
4. Test with Claude Code + Godot
5. Submit a PR with a clear description

## What we need

- New genre presets (with real templates, not stubs)
- Agent improvements (better Godot conventions)
- Bug fixes in hooks
- Documentation improvements

## What we don't accept

- Unity or Unreal content (this is Godot-only)
- Agents that don't serve solo/small-team indie dev
- Features that require paid tools (except GDAI which is optional)
